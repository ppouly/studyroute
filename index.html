StudyRoute

대한민국 학습 패턴 데이터 플랫폼
B2C 플랫폼 교육 데이터 MVP · 강남/송파/분당 Glassdoor 모델
Executive Summary

왜 StudyRoute인가

대한민국 학부모는 세계 최고 수준의 교육열을 가졌지만, 정작 "잘하는 아이들이 어떻게 공부하는가"에 대한 데이터는 없다. 맘카페와 커뮤니티에는 정보가 넘치지만 검증되지 않았고, 워킹맘에겐 접근조차 어렵다. 학원 홍보는 넘쳐나지만 실제 다니는 아이들의 패턴은 보이지 않는다.

StudyRoute는 이 공백을 채운다. 학부모가 직접 입력한 익명 데이터를 기반으로 지역·학년·과목별 실제 학습 패턴을 통계로 제공하는 플랫폼이다. "송파 초3, 영어 잘하는 아이들 — 학원 이용률 84%, 주 평균 3회 (n=19)." 이런 데이터가 한국에 존재한 적이 없었다.

"존재하는 니즈 + 없는 제품" — 수요는 검증됐다. 시장만 기다리고 있다.
The Problem

지금 학부모들이 겪는 것

  • ✗
    정보는 있지만 데이터가 없다. 맘카페, 블로그, 카카오톡 단톡방에 교육 정보는 넘치지만 모두 개인 경험담이다. 집계되지 않고, 검증되지 않으며, 내 상황에 맞는지 알 수 없다.
  • ✗
    워킹맘은 정보 접근 자체가 어렵다. 맘카페 눈팅, 학원 설명회 참석, 주변 엄마들 네트워크 — 이 모든 것에 시간이 있어야 한다. 결정적인 정보 격차가 존재한다.
  • ✗
    학원 정보가 광고로 오염되어 있다. 네이버 검색, 학원 앱, 교육 커뮤니티 — 모두 학원 업체 이해관계가 개입돼 있다. "객관적인" 학원 정보를 신뢰하기 어렵다.
  • ✗
    "잘하는 아이들의 공부 방법" 데이터가 없다. 결과(명문대 합격)는 알지만 과정(어떤 패턴으로 공부했는지)은 아무도 모른다. 의사결정의 근거가 없다.
핵심 페르소나 — 지연, 36세, 맞벌이 워킹맘
송파구 초3 아이를 두고 있다. 수학은 잘하는 것 같은데 영어를 어떻게 시작할지 모른다. 맘카페는 의견이 제각각이고 설명회 갈 시간도 없다. "우리 동네 초3, 영어 잘하는 아이들이 실제로 어떻게 하는지"를 알고 싶다. 지금 그 정보를 얻을 방법이 없다.
The Solution

StudyRoute가 하는 것

부모가 직접 입력한 익명 데이터를 집계해 학습 패턴 통계를 제공한다. 학원 추천 서비스가 아니다. 광고 기반 순위도 아니다. 부모의 데이터를, 부모를 위한 통계로.

데이터 기반 결정

📊 통계 카드 8종

  • 과목 분포 (어떤 과목을 잘하는 아이가 많나)
  • 과목별 주간 시간 비중
  • 수학 패턴 (공부 시간, 학원 이용률, 선행 수준 분포)
  • 영어 / 국어·논술 / 과학 / 코딩·SW 학습 패턴
  • 예체능 패턴 (종목 분포, 주 횟수)
  • 모든 카드에 응답자 수(n) 항상 표시

📍 학원 디렉토리 & 지도

  • 지역·학년 필터 기반 학원 핀 지도
  • 학원별 응답자 수 + 과목 강점 비율
  • 네이버 지도 등록 정보 연결 (등록된 경우)
  • 공공데이터 기반 (광고 아님)
⚡ 설계 원칙: "잘하는 아이들"의 기준은 부모가 과목별로 직접 선택한다. "상위권이냐"고 묻지 않는다. 선행 수준은 수집하지 않는다 — 경쟁·압박 유발 우려. 본인이 자신 있는 과목만 입력 — 편향을 줄이고 설문 피로도를 낮춘다. 모든 통계 카드에는 응답자 수(n)를 항상 표시해 신뢰도를 투명하게 공개한다.

Who This Serves

사용자

🎯 핵심 사용자

초등·미취학 자녀를 둔 부모 (특히 워킹맘). 강남/송파/분당 거주. 교육 결정을 앞두고 있지만 검증된 정보원이 없다.

📤 데이터 제공자

내 아이 공부 정보를 입력하는 부모. 소비자이기도 하다. 1~2분 설문 후 즉시 통계 열람 — 입력이 곧 혜택.

💰 광고주

교육 콘텐츠 사업자, 교재사, 에듀테크. 높은 교육 관심도 타겟. 데이터와 명확히 분리된 광고 슬롯.

Growth Strategy

어떻게 데이터를 빠르게 모을 것인가

StudyRoute의 핵심 과제는 콜드 스타트다. 데이터가 없으면 사람이 안 오고, 사람이 없으면 데이터가 안 쌓인다. 두 가지 메커니즘으로 이 문제를 푼다.

📬 개인 리포트 인센티브

"설문 입력하면 주변 파악 리포트를 드려요"

  • 리포트 목적: 내 아이 확인이 아닌 "주변 파악"
  • 이메일로는 티저만 발송 → 풀 리포트는 서비스에서 열람
  • n<15이면 대기 화면 + 예상 완료 기간 표시
  • n≥15 달성 시 이메일로 재유입 트리거

🔗 공유 바이럴 루프

공유 수에 따라 열람 권한이 확장되는 구조

  • 3명 공유 완료 → 전 과목 통계 + 타 과목 잘하는 아이 패턴
  • 5명 공유 완료 → 타 지역 데이터 비교
  • 맘카페/카카오 공유 행동 패턴에 최적화
  • ref 파라미터 추적 → 카운트 자동 업데이트

MVP 범위

✅ MVP에 포함

  • 통계 서비스: 강남 / 송파 / 판교
  • 설문 수집: 전국 시도구 (공유 유입 전국 대응)
  • 비서비스 지역 → "순차 오픈 예정" 안내 + 서비스 지역 통계 미리보기 1개
  • 미취학·초등 대상
  • 무로그인 설문 (기기 토큰 발급)
  • 통계 카드 8종 (수학/영어/국어·논술/

  • 개인 리포트 (내 아이 포지셔닝 + 타 과목 패턴)

  • n<15 대기 화면 + 예상 완료 기간 표시
  • 공유 티어 (3명→전과목, 5명→타지역)
  • 선택적 이메일 수집 + n≥15 알림
  • 로그인 전환 (공유 완료 후 / 이메일 재방문)
  • 학원 지도 (공공데이터 + 카카오 지오코딩)

  • 광고 슬롯 (데이터와 명확히 구분)

Revenue Model

수익 구조

MVP는 광고 단일 모델. 통계 카드·학원 지도·개인 리포트 모두 무료. 프리미엄 과금은 MVP 이후 검토 (다자녀 관리, 학기별 추적 등).

📰 네이티브 광고

피드 중간 삽입. 교육 콘텐츠/서비스 광고. 데이터 카드와 시각적으로 명확히 구분.

🖼 배너 광고

교육 관련 브랜드 노출. 학기 초/입학 시즌 단가 프리미엄 기대.

핵심 원칙: 광고가 데이터 통계에 어떠한 영향도 미치지 않는다. 학원 업체의 광고비가 통계 순위에 반영되는 구조 일절 금지.

Vision

StudyRoute가 3년 후 되고 싶은 것

지금

강남/송파/분당 학습 패턴 데이터 허브
