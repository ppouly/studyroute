StudyRoute

대한민국 학습 패턴 데이터 플랫폼
B2C 플랫폼 교육 데이터 MVP · 강남/송파/분당 Glassdoor 모델
Executive Summary

왜 StudyRoute인가

대한민국 학부모는 세계 최고 수준의 교육열을 가졌지만, 정작 "잘하는 아이들이 어떻게 공부하는가"에 대한 데이터는 없다. 맘카페와 커뮤니티에는 정보가 넘치지만 검증되지 않았고, 워킹맘에겐 접근조차 어렵다. 학원 홍보는 넘쳐나지만 실제 다니는 아이들의 패턴은 보이지 않는다.

StudyRoute는 이 공백을 채운다. 학부모가 직접 입력한 익명 데이터를 기반으로 지역·학년·과목별 실제 학습 패턴을 통계로 제공하는 플랫폼이다. "송파 초5, 수학 잘하는 아이들 — 주 평균 공부 시간 2.3시간, 선행 1학기 이상 68%." 이런 데이터가 존재한 적이 없었다.

"존재하는 니즈 + 없는 제품" — 수요는 검증됐다. 시장만 기다리고 있다.
The Problem

지금 학부모들이 겪는 것

  • ✗
    정보는 있지만 데이터가 없다. 맘카페, 블로그, 카카오톡 단톡방에 교육 정보는 넘치지만 모두 개인 경험담이다. 집계되지 않고, 검증되지 않으며, 내 상황에 맞는지 알 수 없다.
  • ✗
    워킹맘은 정보 접근 자체가 어렵다. 맘카페 눈팅, 학원 설명회 참석, 주변 엄마들 네트워크 — 이 모든 것에 시간이 있어야 한다. 결정적인 정보 격차가 존재한다.
  • ✗
    학원 정보가 광고로 오염되어 있다. 네이버 검색, 학원 앱, 교육 커뮤니티 — 모두 학원 업체 이해관계가 개입돼 있다. "객관적인" 학원 정보를 신뢰하기 어렵다.
  • ✗
    "잘하는 아이들의 공부 방법" 데이터가 없다. 결과(명문대 합격)는 알지만 과정(어떤 패턴으로 공부했는지)은 아무도 모른다. 의사결정의 근거가 없다.
핵심 페르소나 — 5세 자녀를 둔 워킹맘
영어유치원 입학을 앞두고 있다. 주변 엄마들에게 물어봤지만 의견이 제각각이다. 직접 학원 설명회를 다닐 시간이 없고, 맘카페 정보는 믿기 어렵다. "우리 동네 영어유치원 중에 실제 아이들이 많이 다니는 곳이 어딘지, 다니고 나서 어떤지"를 알고 싶다. 지금 그 정보를 얻을 방법이 없다.
The Solution

StudyRoute가 하는 것

부모가 직접 입력한 익명 데이터를 집계해 학습 패턴 통계를 제공한다. 학원 추천 서비스가 아니다. 광고 기반 순위도 아니다. 부모의 데이터를, 부모를 위한 통계로.

데이터 기반 결정

📊 학습 패턴 통계

  • 과목별 잘하는 아이들의 공부 시간 분포
  • 선행 수준 분포 (수학 몇 학기 선행?)
  • 학원 vs 자기주도 비율
  • 영어유치원 → 이후 영어 유지 비율

📍 위치 기반 학원 디렉토리

  • 내 위치 기준 근처 학원 목록
  • 학원별 실제 응답자 수 표시
  • 영어유치원 전용 필터
  • 공공데이터 기반 (광고 아님)
⚡ 설계 원칙: "잘하는 아이들"의 기준은 부모가 과목별로 직접 선택한다. "상위권이냐"고 묻지 않는다. 본인이 자신 있는 과목만 입력 — 편향을 줄이고 설문 피로도를 낮춘다. 모든 통계 카드에는 응답자 수(n)를 항상 표시해 신뢰도를 투명하게 공개한다.
Differentiation

왜 이 접근이 다른가

기준 맘카페/커뮤니티 학원 앱/플랫폼 StudyRoute
데이터 출처 개인 경험담 학원 제공 부모 직접 입력
광고 영향 간접적 직접 (광고비 순위) 데이터와 완전 분리
집계/통계 없음 없음 핵심 가치
과정 데이터 결과 중심 없음 공부 방법/시간/패턴
접근성 네트워크 필요 누구나 입력 → 즉시 열람

참고 모델: 잡플래닛/블라인드 — 사용자 제공 데이터로 신뢰 구축, 광고는 별도 영역에서 수익화.

Who This Serves

사용자

🎯 핵심 사용자

초등·미취학 자녀를 둔 부모 (특히 워킹맘). 강남/송파/분당 거주. 교육 결정을 앞두고 있지만 검증된 정보원이 없다.

📤 데이터 제공자

내 아이 공부 정보를 입력하는 부모. 소비자이기도 하다. 1~2분 설문 후 즉시 통계 열람 — 입력이 곧 혜택.

💰 광고주

교육 콘텐츠 사업자, 교재사, 에듀테크. 높은 교육 관심도 타겟. 데이터와 명확히 분리된 광고 슬롯.

Growth Strategy

어떻게 데이터를 빠르게 모을 것인가

StudyRoute의 핵심 과제는 콜드 스타트다. 데이터가 없으면 사람이 안 오고, 사람이 없으면 데이터가 안 쌓인다. 두 가지 메커니즘으로 이 문제를 푼다.

📬 리포트 인센티브

"설문 입력하면 내 아이 비교 리포트를 드려요"

  • 추상적인 "통계 보기"보다 구체적인 가치 제안
  • 이메일 수집 → CRM 자산으로 전환
  • 학기마다 재입력 → 업데이트 리포트 발송

🔗 공유 바이럴 루프

3명 공유 시 상세 리포트 즉시 제공

  • 맘카페/카카오 공유 특성 최대 활용
  • "나 이거 써봤는데 좋더라" 신뢰 기반 확산
  • 공유자 추적 → 리포트 혜택 자동 지급
리포트 발송 2단계 전략
Phase 1 (초기, 데이터 부족): "📬 리포트 1순위 예약 완료 — 송파 초5 수학 데이터가 모이면 가장 먼저 받아요"
Phase 2 (n≥15 충족): 즉시 개인화 리포트 발송
→ 초기 참여자는 "우리 지역 데이터를 처음 만드는 사람"으로 포지셔닝 — 기대감과 소속감 동시에

📊 리포트 샘플

송파구 초5 · 수학 잘하는 아이들 비교 (n=23)
우리 아이 주 공부 시간: 2시간  |  같은 조건 평균: 2.3시간
우리 아이 선행 수준: 1학기  |  같은 조건 상위 30%: 2학기 이상
학원 이용: 주 3회  |  같은 조건 평균: 주 2.8회

목표

  • 설문 완료 후 통계 페이지 체류 시간
  • 재방문율 (학기 초 급증 여부)

Scope

MVP 범위

✅ MVP에 포함

  • 강남 / 송파 / 분당 지역
  • 미취학·초등 대상
  • 1~2분 설문 (잘하는 과목 선택형)
  • 학원 디렉토리 (공공데이터 + 지오코딩)
  • 통계 카드 (n수 항상 표시)
  • n<5 완전 비공개 (프라이버시)
  • 기기 기반 중복 방지
  • 개인정보처리방침 + 동의 체크박스
  • 광고 슬롯 (데이터와 명확히 구분)
  • 리포트 예약 + 이메일 수집
  • 3명 공유 시 상세 리포트 지급
  • n≥15 달성 시 예약자 자동 발송

❌ MVP 이후로 연기

  • 전국 지역 확장
  • 중학교 / 고등학교
  • 소셜 로그인 / 계정 시스템
  • 세그먼트 자동 롤업
  • 종단 데이터 추적 (연간 변화)
  • B2B 데이터 라이선싱
  • 앱 출시 (웹 우선)
⚡ 치킨앤에그 전략: MVP 초기 100명은 통계 카드 노출이 어렵다 (지역×학년 분산 시 n<15). 초기에는 구글폼으로 데이터 먼저 수집 → 300~500명 확보 후 플랫폼 오픈. 맘카페·당근 커뮤니티 집중 공략으로 특정 세그먼트(송파 초등 수학 등) 먼저 채운다.
Revenue Model

수익 구조

📰 네이티브 광고

피드 중간 삽입. 교육 콘텐츠/서비스 광고. 데이터 카드와 시각적으로 명확히 구분.

🖼 배너 광고

교육 관련 브랜드 노출. 학기 초/입학 시즌 단가 프리미엄 기대.

📦 데이터 라이선싱 (이후)

익명 집계 데이터를 교육 연구기관·출판사에 제공. 개인 식별 불가 수준.

핵심 원칙: 광고가 데이터 통계에 어떠한 영향도 미치지 않는다. 학원 업체의 광고비가 통계 순위에 반영되는 구조 일절 금지.

Vision

StudyRoute가 3년 후 되고 싶은 것

지금

강남/송파/분당 학습 패턴 데이터 허브

특정 지역 학부모 커뮤니티의 신뢰를 얻는다. 맘카페 대신 StudyRoute를 여는 습관을 만든다.

1년

수도권 전역 + 종단 데이터 확보

"영어유치원 다닌 아이, 초등 3학년 때 영어 수준은?" — 시간 흐름에 따른 패턴 데이터가 생기기 시작한다.

3년

대한민국 학습 패턴 데이터베이스

학부모가 교육 결정을 데이터로 내리는 것이 당연해지는 세상. 교육 연구기관이 StudyRoute 데이터를 인용한다.
